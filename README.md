## 功能列表

  * bookspace, 类似于workspace的概念，方便多本书之间的管理
    * list books
    * create book
    * edit book property
    * delete book

  * bookedit
    * open book
    * menu (create/delete/edit/order chapter)
    * edit file and save file
    * preview file

  * tools
    * theme
    * export
      * export as pdf
      * export as web site
    * git
      * commit
      * logs
