# Mbook快速上手

## 目录管理

目录区域支持鼠标右键，可以通过右键快速的：

* 新建章节
* 新建子章节
* 修改名称
* 删除章节

同时还可以通过拖拽来调节章节顺序。

## 编辑区

编辑区在中间一列， 可以编辑输入markdown语法。

目前对表格的支持还不太友好，后续会增加方便的表格数据编辑器

## 预览区

预览区在最右侧一列，可以通过快捷键`cmd + p` 来快速开关

## 快捷键

```
cmd + p     // 切换预览区
option + x  // 截图功能
cmd + v     // 黏贴图片， 黏贴图片支持常见聊天工具截图，以及各种图形编辑器，可以直接 cmd + c , cmd + v 快速复制进入
```
